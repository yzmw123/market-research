# Standard Research

## 适用范围

用于市场机会、进入判断、竞品、产品方向、招投标格局、公司公开尽调和市场空间研究。

## 选择章节

根据证据地图从下列模块中选择 4-7 个，不要求全部使用：

| 模块 | 适用问题 | 典型证据 |
|---|---|---|
| 定义与边界 | 概念容易漂移 | 政策、标准、分类、产品定义 |
| 市场与增长 | 是否有空间 | 统计、财报、采购、可信报告 |
| 需求与采购 | 谁买、为何买、怎么买 | 招标、案例、评论、访谈 |
| 政策与标准 | 预算或准入受监管驱动 | 政策、预算、标准、处罚案例 |
| 产业链与价值链 | 谁获利、卡点在哪里 | 企业业务结构、成本和毛利 |
| 竞争格局 | 玩家和生态位 | 财报、产品、融资、投标和案例 |
| 技术与产品演进 | 机会窗口受技术变化影响 | 文档、版本、招聘、专利和论文 |
| 商业模式与交付 | 是否可复制 | 定价、渠道、合同、案例和访谈 |
| 进入机会与风险 | 最终决策 | 前文证据、盲区和验证计划 |

## 常见结构

### 市场机会

研究边界 → 市场与政策信号 → 需求触发 → 供给和竞争 → 可进入切口 → 风险盲区 → 建议。

### 产品与竞品

用户任务和替代方案 → 竞品定位与能力边界 → 体验、定价和渠道 → 差异化空白 → 验证方案。

竞争包括人工流程、Excel、旧系统、外包和“不做决定”，不只包括直接竞品。

### ToG 与招投标

政策和资金 → 项目类型和金额 → 供应商角色 → 技术与准入 → 公开信息盲区 → 进入验证。

公开资料不能证明内部评标、专家关系、暗价和未公开准入规则。

### 公司公开尽调

公司和股权 → 管理层 → 经营和财务 → 产品与客户 → 法律和公共风险 → 行业位置 → 合作风险。

## 市场规模

只有在数据影响决策时才做 TAM/SAM/SOM。优先同时给出自上而下和自下而上估算，使用区间而非伪精确单点，并标注每个假设。

## 盲区转化

对决策关键但公开资料不可得的变量，说明：

1. 为什么重要。
2. 检查过哪些代理信号。
3. 为什么仍不能回答。
4. 用访谈、POC、问卷、付费数据或内部数据如何验证。
